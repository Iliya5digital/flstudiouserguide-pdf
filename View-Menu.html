Окна
Список воспроизведения ( F5 ) — Показать/скрыть окно списка воспроизведения .
Piano roll ( F7 ) — показать/скрыть окно Piano roll .
Стойка каналов ( F6 ) — показывает/скрывает окно Стойки каналов и пошагового
секвенсора .
Микшер ( F9 ) — показать/скрыть окно микшера .
Браузер ( Alt+F8 ) — показать/скрыть окно браузера .
Средство выбора проекта ( Ctrl + F8 ) — отображает массив клипов и каналов ,
используемых в проекте, и взаимосвязь между ними, обратите внимание на желтые
галочки, когда курсор мыши находится над элементом. Основная цель средства
выбора проекта — выбрать элементы, щелкнув их, для использования в списке
воспроизведения.
Выбор плагинов ( F8 ) — отображает массив плагинов ( эффекты слева, инструменты
справа). Чтобы использовать средство выбора подключаемых модулей, нажмите,
удерживайте и перетащите значок, связанный с подключаемым модулем, который вы
хотите использовать, и список средств выбора закроется. Поместите плагин в нужное
место внутри FL Studio. См. раздел руководства « Браузер » для получения подробной
информации о настройке средства выбора плагинов.
Tempo tapper - управление темпом Open Tap .
Сенсорный контроллер ( Alt+F7 ) — просмотр текущего выбранного сенсорного
контроллера .
Вывод сценария . Вкладка « Интерпретатор » используется для тестирования команд
Python MIDI Scripting . Вторая вкладка . Если в настройках MIDI выбран MIDI-скрипт,
появится вторая вкладка с именем устройства (в данном случае FL STUDIO FIRE).
Открывайте и редактируйте сценарии с помощью сценария Edit и выполняйте их с
помощью кнопок Reload script .
Панели инструментов — параметры включают:
Редактировать — открывает « меню пресетов » на панели инструментов. Загружайте,
сохраняйте и редактируйте пресеты панели инструментов.
Разделители - Добавить/удалить линии между панелями.
Панель подсказок — открывает панель расширенных подсказок .
Плоские кнопки — добавьте/удалите трехмерное затенение вокруг кнопок.
Скрыть ( Ctrl+F11 ) — скрыть панель инструментов, чтобы получить больше места по
вертикали.
Тесты — открывает окно «Тесты», которое позволяет запускать тесты по запросу
службы технической поддержки. Используется для устранения неполадок в системах.
Журналы тестирования сохраняются в ..\[user]\Documents\Image-Line\Data\FL
Studio\Logs\Test mode
Монитор производительности плагинов — используйте этот инструмент, чтобы найти
плагины в вашем проекте, использующие больше всего ЦП, по мере того, как вы
определяете причину опустошения . Показанные значения представляют собой время
обработки в микросекундах или микросекундах ( 1/1 000 000 секунды). Когда значение
превышает 100 000 мкс (0,1 секунды), оно будет отображаться в секундах.
ПРИМЕЧАНИЕ. Это окно можно открыть, дважды щелкнув панель ЦП .
Сброс при транспортировке — сброс сохраненных (пиковых) значений команд
воспроизведения/остановки.
Заморозить — приостанавливает анализ, чтобы вы могли хмуро взглянуть на данные.
Столбцы:
Плагин — название плагина и патча (если доступно).
Тип — инструмент (Synth) против эффекта (FX).
% - Процент звукового буфера, необходимый для обработки плагина. Подробное
объяснение этого значения см. в разделе Панель ЦП .
Время — последнее записанное измерение времени обработки для подключаемого
модуля (не все подключаемые модули активны все время). Значение представляет
собой процессорное время в секунду.
Пик — максимальное измеренное время обработки.
Total - Сумма всех измерений времени для этого плагина. Это позволяет сравнивать
обработку по всем проектам.
Найти — отфильтровать определенные плагины, введя полное название или его часть.
Макет
Закрыть все окна ( F12 ) — закрывает все окна в FL Studio.
Закрыть все окна плагинов ( Alt+F12 ) — закрывает только все окна плагинов.
Close all unfocused windows ( Ctrl+F12 ) — Закрывает все невыбранные (не
сфокусированные) окна.
Выровнять все редакторы каналов ( Shift + F12 ) — упорядочивает все окна плагинов
каналов, чтобы выровнять их по левому верхнему углу выделенного плагина.
ПРИМЕЧАНИЕ. Это не будет работать с «Отдельными» плагинами.
Расставить окна
Realign ( Shift+Ctrl+H ) — восстанавливает макет установки по умолчанию.
В рабочую область — перемещает все окна в видимую рабочую область рабочего
стола FL Studio. Полезно то, что окна были перемещены за пределы рабочей области.
Например, когда использовался второй монитор, который теперь недоступен).
Рабочий стол (по умолчанию) — восстанавливает макет по умолчанию.
Альтернативный — просто альтернативный макет. Попробуйте, может понравится :)
Планшет (по умолчанию) — удобный для сенсорного экрана макет с важными
элементами панели инструментов, расположенными вдоль нижней части окна.
Планшет (альтернативный) - Альтернативный вариант макета планшета.
Пользовательский — восстанавливает пользовательский макет. Используйте опцию
Сохранить текущую аранжировку ниже, чтобы создать ее.
Сохранить пользовательский макет — сохраняет текущий макет. Полезно перед
изменением разрешения экрана (т. е. изменением размера экрана или поворотом
экрана планшетного ПК).
Фон — позволяет выбрать обои для главного окна FL Studio.
Установить изображение обоев — позволяет использовать изображение в качестве
фона главного окна.
Установить динамические обои / HTML-документ — позволяет выбрать динамические
обои или HTML-документ в качестве фона главного окна. Динамические обои в
формате .txt можно редактировать с помощью стандартного текстового редактора. Они
отображают информацию о версии FL Studio, размываются, когда FL Studio не
сфокусирована, и настраиваются в соответствии с настройками масштабирования .
Установить цвет фона - изменить цвет фона рабочего стола.
Браузер
История отмены — раскрывает раздел истории в Диспетчере проектов .
Автоматизация — расширяет раздел истории в Диспетчере проектов .
Используемые генераторы — расширяет раздел инструментов в Диспетчере проектов .
Используемые эффекты — расширяет раздел эффектов в Диспетчере проектов .
Удаленное управление — расширяет раздел удаленного управления в Диспетчере
проектов .
База данных плагинов — открывает базу данных плагинов ( Shift+F8 ).
