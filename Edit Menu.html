Меню «Правка» содержит команды для выбора и редактирования каналов, нот и
клипов (цель меню будет зависеть от того, какое окно в FL Studio выбрано).
Команды
Отменить ( Ctrl+Alt+Z / Ctrl+Z ) — эти две команды позволяют вам пройтись по истории
редактирования FL Studio (чтобы увидеть список действий, хранящихся в данный
момент в истории, откройте папку Project Browser /History). Ctrl+Alt+Z позволяет
вернуться назад по истории. Ctrl+Z выполняет переход вперед по истории, если только
вы не находитесь на самом последнем шаге (в этом случае Ctrl+Z работает как
стандартное сочетание клавиш отмены/повтора на один шаг).
Вырезать ( Ctrl+X ) — вырезает ноты из всех выбранных каналов в пошаговом
секвенсоре в буфер обмена. События автоматизации не могут быть вырезаны с
помощью этой команды.
Копировать ( Ctrl+C ) — копирует содержимое всех выбранных каналов в пошаговом
секвенсоре в буфер обмена. События автоматизации не могут быть скопированы с
помощью этой команды.
Вставить ( Ctrl+V ) — вставляет ноты из буфера обмена, начиная с самого верхнего
выбранного канала в пошаговом секвенсоре .
