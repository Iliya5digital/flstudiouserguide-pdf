Поддержка Apple Silicon — FL Studio теперь может открывать плагины Intel VST и AU

Аудио запись:
Список воспроизведения - элементы управления на дорожке: вход, место записи,
мониторинг и постановка на охрану. В меню «Плейлист» появилась новая опция для
отображения/скрытия элементов управления «Вид > Показать элементы управления
на аудиодорожках».
Место записи — выберите одно из 6 мест на пути входного сигнала, включая прямой
вход аудиоинтерфейса. При выборе внешнего входа для места записи будет
установлено значение « Только внешний вход », что позволяет избежать записи
внутреннего звука. Полезно при создании нескольких дублей одной и той же
аудиодорожки списка воспроизведения / вставки микшера.
Новые параметры монитора (Выкл., При охране и Вкл.) — из списка воспроизведения ,
( щелчок правой кнопкой мыши) заголовок дорожки списка воспроизведения,
аудиодорожки > режим дорожки > выберите параметр аудиовхода , чтобы выбрать
аудиовход. Значение по умолчанию для параметра « Контроль внешнего входа » равно
« При постановке на охрану ».
Циклическая запись — повышена стабильность благодаря небольшим циклам.
Треки микшера — теперь можно активировать трек микшера, для которого не выбран
внешний аудиовход.
Удаленные записи — улучшена обработка « помещенных в корзину записей ».
Автоматизация:
Редактор клипов автоматизации — новый редактор автоматизации. Включает в себя
список целей и возможность управлять, редактировать и находить цели ссылок . Также
существует множественный выбор точек во всех редакторах конвертов (включая
плагины).
Редактор событий > Клип автоматизации — новый алгоритм преобразования более
точно представляет исходные данные событий с меньшим количеством контрольных
точек и аппроксимацией кривой.
Автоматизация клипов и огибающих — ( Shift+щелчок правой кнопкой мыши ) для
добавления контрольных точек при текущем значении огибающей. Работает в
плейлисте, редакторе клипов автоматизации и конвертах плагинов (например, Sytrus,
Harmor и т. д.). ПРИМЕЧАНИЕ. Если привязка включена , используйте (
Alt+Shift+щелчок правой кнопкой мыши )
Стойка каналов и каналы:
Каналы слоев — добавлен режим последовательного воспроизведения (циклический
стиль).
Селекторы стойки каналов — для выбранного канала изменено с ( Одиночный щелчок
) на ( Двойной щелчок ), чтобы выбрать все каналы.
Настройки канала - ( Правая кнопка + щелчок ) « Введите значение » Управление
диапазоном изменения высоты тона канала теперь допускает полутоновые значения .
Смеситель:
Значения подсказок для ручек посыла микшера теперь показывают значения дБ.
Меню > Выбранная дорожка (дорожки) > Назначить новой звуковой дорожке
(дорожкам) '- Чтобы назначить выбранные дорожки микшера в список
воспроизведения и создать звуковые дорожки для каждой.

Плейлист и пианоролл:
Элементы управления записью на дорожке — для аудиодорожек те же функции, что и
на панели дорожек микшера.
Пиковые измерители для аудиодорожек — дополнительный пиковый измеритель для
аудиодорожек. Включите или отключите в списке воспроизведения « Меню> Вид>
Показать уровни на аудиодорожках ».
Выбор и множественный выбор — теперь можно выбирать дорожки плейлиста. (
Ctrl+стрелки вверх/вниз ) выбирает дорожку. ( Ctrl+Alt+стрелка вверх/вниз ) расширяет
выделение.
Клонирование дорожек ( щелчок правой кнопкой мыши ) — используйте меню
заголовка дорожки , чтобы дублировать существующие дорожки списка
воспроизведения с возможностью клонирования паттернов , аудиоклипов ,
автоматизации , эффектов (в связанных дорожках микшера) и сгруппированных
дорожек списка воспроизведения .
Множественная загрузка — удаление предустановки инструмента на несколько
выбранных дорожек плейлиста добавит экземпляр плагина ко всем выбранным
тактам. Если дорожки находятся в режиме «Инструментальная дорожка» —
перетаскивание предустановки эффекта на одну из нескольких выбранных
инструментальных дорожек добавит эффект ко всем выбранным дорожкам в группе.
Сделать уникальными клипы — теперь отображается предупреждение, когда вы
выбираете плейлист и используете « Сделать уникальными », если выбрано большое
количество клипов. Ты уверен? [Д][Н].
Объединить клипы — теперь работает для всех выбранных типов клипов.
Контраст сетки — «Плейлист и пианоролл», «Просмотр» > «Контрастность сетки» >
«Высокий», «Средний» или «Низкий».
