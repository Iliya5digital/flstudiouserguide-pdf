Меню «Файл» ( F3 ) содержит команды для управления файлами
Команды
Новый — запуск нового пустого проекта с использованием последнего выбранного
шаблона в качестве отправной точки. Вам будет предложено сохранить любые
изменения, внесенные в текущий проект.
Новое из шаблона — Содержит подменю с выбором различных шаблонов — Жанр ,
Минимальный , Другой и Полезный . Чтобы установить шаблон по умолчанию при
открытии новых проектов, используйте параметр « Общие настройки» > «Шаблон по
умолчанию ».
Открыть ( Ctrl+O ) — открывает существующий проект. Диалоговое окно «Открыть
файл» дополнено разделом « Избранное» .
Сохранить ( Ctrl+S ) — обновляет проект, перезаписывая последнюю сохраненную
копию на диске. Если текущий проект не был сохранен, перед сохранением откроется
диалоговое окно имени. Список доступных форматов сохранения см. на странице
Форматы файлов .
Сохранить как ( Shift+Ctrl+S ) — сохраняет текущий под другим именем и/или
форматом. Список доступных форматов сохранения см. на странице Форматы файлов
. Диалоговое окно «Сохранить файл» дополнено разделом « Избранное» .
Сохранить новую версию ( Ctrl+N ) — сохраняет текущий проект в виде файла с
уникальным именем. Например, если у вас есть проект с именем MyProj , эта команда
сохраняет файл с именем MyProj_2 . В следующий раз он будет называться MyProj_3 и
т. д . Эта функция полезна для сохранения версий проекта без перезаписи исходного
файла.
Импорт — загружает файлы, которые не являются собственными проектами FL Studio.
Полный список типов файлов, которые вы можете импортировать, см. на странице
Открытие/импорт форматов файлов . Диалоговое окно «Импорт файла» дополнено
разделом « Избранное» .
Экспорт - Преобразует текущий проект в различные форматы файлов. Полный список
типов файлов, в которые можно экспортировать, см. на странице Форматы файлов .
Файл волны ( Ctrl+R ) — формат волны — это широко используемый несжатый
аудиоформат, используемый для распространения звука производственного качества.
Если вы планируете записать стандартный аудио компакт-диск, вам потребуется
экспортировать файлы 16-битной волны 44,1 кГц (.wav). ПРИМЕЧАНИЕ. FL Studio не
записывает компакт-диски.
Файл mp3 ( Shift+Ctrl+R ) — mp3 (Mpeg Audio Layer 3) — это сжатый аудиоформат,
полезный для обмена музыкой через Интернет и совместимые mp3-плееры. Хотя
качество звука в формате .mp3 может быть превосходным (при скоростях 224 кбит/с и
выше), его не рекомендуется использовать для архивирования звука
производственного качества.
ogg-файл — ( Shift+Ctrl+R ) — ogg (Ogg Vorbis) — это сжатый аудиоформат с открытым
исходным кодом, полезный для обмена музыкой через Интернет и совместимые
проигрыватели Ogg. Хотя качество звука Ogg может быть превосходным (при скорости
224 кбит/с и выше), его не рекомендуется использовать для архивирования звука
производственного качества.
Файл MIDI ( Shift+Ctrl+M ) — формат MIDI (цифровой интерфейс музыкальных
инструментов) сохраняет музыкальные ноты (партитуры) и управляющие данные,
связанные с проектами. ПРИМЕЧАНИЕ. MIDI не содержит и не сохраняет аудио. Ваш
компьютер может издавать звук при запуске MIDI-файла, поскольку многие
аудиоустройства могут интерпретировать и воспроизводить MIDI-данные.
Загрузить в облако — предоставляет возможность загружать аудио на различные
веб-сайты социальных сетей.
Все дорожки плейлиста — последовательно преобразует каждую дорожку плейлиста в
аудиофайл. Существует также функция « Разделить дорожки микшера» , которая
делает то же самое, но создает отдельный аудиофайл для каждой дорожки микшера.
ПРИМЕЧАНИЯ. Прежде чем использовать эту функцию , обратите внимание на
диалоговое окно «Визуализация» > «Разное» > «Включить мастер-эффекты» . Если вы
собираетесь использовать дорожки списка воспроизведения в другой DAW, возможно,
вы также захотите включить мастер-эффекты. Подробную информацию об этой
функции см. в разделе « Переключение на аудио/замораживание ». Есть некоторые
предостережения, связанные с ожидаемым рендерингом эффектов микшера .
С начала песни — создайте звуковую дорожку для каждой дорожки плейлиста с начала
плейлиста.
С начала дорожки — создайте звуковую дорожку для каждой дорожки списка
воспроизведения с начала данных в дорожке списка воспроизведения. Это приведет к
неравномерной длине «стеблей» (файлов).
Выбор времени — создайте звуковую дорожку для каждого списка воспроизведения
для выбранного диапазона времени.
Все группы дорожек плейлиста — то же, что и выше, но создает только один
аудиофайл для сгруппированных дорожек плейлиста (аудио родительской дорожки и
поддорожки объединены).
Заархивированный циклический пакет — сохраняет как .flp (данные проекта), так и
образцы, связанные с проектом, в стандартном формате файла ZIP . Это отличный
способ заархивировать важные проекты, чтобы не потерять связанные с ними звуки.
Кости проекта — используйте эту команду в качестве ярлыка для экспорта всех
настроек генератора/эффектов в виде пресетов ( файлы *.fst ) и данных
партитуры/автоматизации в виде файлов партитуры (файлы *.fsc ), используемых в
текущем проекте. Это полезно для объединения компонентов из одного проекта в
другой. Кроме того, в выбранной вами папке будет создан файл *.nfo для хранения
данных пресета/партитуры, который используется браузером .
Файлы данных проекта — сохраняет копии всех сэмплов, пресетов и других данных
плагинов, используемых в текущем проекте, в одну папку по вашему выбору.
Параметры командной строки — см. здесь .
Недавние проекты ( Alt + 1 до 0 (10) ) — отображается в виде списка из 10 последних
открытых проектов в нижней части меню «Файл», причем самый последний находится
вверху.
Вернуться к последней резервной копии — загружает последнюю резервную копию
(см. Настройки резервного копирования ).
Выход — закрывает FL Studio, предлагая вам сохранить любые изменения в текущем
проекте.
